Why Java is Object Oriented Programming?

Why Java is Object Oriented Programming? Java is purely object oriented programming language because without class and object it is impossible to write any Java program. Java is not pure object oriented programming language. because java supports non-primitive datatypes like int ,float ,boolean,double,long etc.
